# 내 장부

소상공인을 위한 간편 수입/지출 장부 앱. 세무사에게 넘기기 전까지 방치하지 않게 도와주는 게 목표.

React + TypeScript + Vite로 만든 클라이언트 앱. Supabase(PostgreSQL)로 클라우드 저장, 이메일 매직링크로 로그인.

---

## 구현된 기능

### 인증
- 이메일 주소 입력 → 매직링크 발송 → 클릭 한 번으로 로그인
- 비밀번호 없음, 기기 간 데이터 공유
- 기존 localStorage 데이터 첫 로그인 시 자동 마이그레이션

### 온보딩 (3단계 마법사)
- **Step 1** — 업종 선택 + 사업자 유형(일반/간이과세자) + 직원 유무 설정
- **Step 2** — 첫 거래 바로 입력 (건너뛰기 가능)
- **Step 3** — 매달 고정 지출 반복 거래 등록 (건너뛰기 가능)

### 거래 입력 (`TransactionForm`)
- 수입 / 지출 구분 (세그먼트 컨트롤)
- 금액, 날짜, 거래처, 메모, 항목 입력
- **카테고리 자동 제안** — 거래처·메모 키워드 기반으로 항목을 추천해 원클릭 선택
- **반복 거래 등록** — 매월 / 매주 선택 후 체크 한 번으로 자동화
- **최근 거래 복제** — ↩ 복제 버튼으로 직전 거래 내용을 폼에 자동 채움
- **템플릿 저장/적용** — 자주 쓰는 거래를 템플릿으로 저장, 기본 제공 템플릿 포함
- **거래 수정 모드** — 내역에서 수정 요청 시 폼이 수정 모드로 전환
- **메모 빈칸 힌트** — 메모 없이 저장 시 한 줄 남기도록 유도, "그래도 저장" 선택 가능
- **초심플 카테고리 모드** — 원가 / 광고비 / 사무용품 3가지로 단순화

### 거래 내역 (`TransactionList`)
- **기간 필터** — 이번달 / 지난달 / 전체 / 기간 직접 선택
- **수입/지출 필터** + **거래처·메모 텍스트 검색**
- 날짜별 그룹핑, 일별 소계 표시
- 기간 합계 스트립 — 건수, 수입 합계, 지출 합계, 순이익
- 거래 항목 클릭 시 상세 펼치기 → 수정 / 복제 / 삭제
- 반복 거래 아이콘 🔄 표시
- **CSV 가져오기** — UTF-8 CSV 업로드
- **CSV 내보내기** — 홈택스 간편장부 형식으로 다운로드

### 요약 (`Summary`)
- 월 네비게이터 — 년/월 드롭다운 또는 ‹ › 버튼으로 이동
- **월별 KPI 카드** — 수입, 지출, 순이익
- **지출 항목 비율 차트** — 카테고리별 바 그래프
- **연간 현황** — 연 총 수입, 지출, 순이익
- **세금 추정 (참고용)** — 단순경비율 60% 기준 종합소득세 추정치
- **반복 거래 관리** — 등록된 규칙 목록, 일시정지 / 재개 / 삭제

### 세무 일정 체크리스트 (`Checklist`)
- 사업자 유형(일반/간이과세자) + 직원 유무에 따른 맞춤 항목 표시
- D-day 배지 (D-7 이내 강조)
- 월별 체크 상태 Supabase 저장

### 반복 거래 자동화
앱을 열 때 마지막 실행 이후 누락된 반복 거래를 모두 소급 생성.

### UX
- 토스트 알림 (저장/수정/삭제 피드백)
- **반응형 레이아웃** — 모바일: 하단 탭 네비게이션 / 데스크톱: 왼쪽 사이드바
- 헤더에 이번달 순이익 상시 표시

## 프로젝트 구조

```
src/
├── App.tsx                  # 루트 — 인증, 탭 상태, 거래 CRUD, 반복 자동화
├── types/
│   └── transaction.ts       # Transaction, Template, RecurringRule, Industry 타입
├── lib/
│   ├── supabase.ts          # Supabase 클라이언트
│   └── db.ts                # 전체 CRUD 비동기 함수 + localStorage 마이그레이션
├── components/
│   ├── Login.tsx            # 이메일 매직링크 로그인 화면
│   ├── Onboarding.tsx       # 3단계 온보딩
│   ├── TransactionForm.tsx  # 입력 폼 (추가/수정 겸용)
│   ├── TransactionList.tsx  # 내역 목록 + 필터 + CSV
│   ├── Summary.tsx          # 요약 + 세금 추정 + 반복 관리
│   └── Checklist.tsx        # 사업자 유형별 맞춤 세무 일정 체크리스트
└── utils/
    ├── storage.ts           # localStorage 읽기/쓰기 (마이그레이션 전용)
    ├── categoryPredict.ts   # 카테고리 자동 제안 로직
    ├── taxEstimate.ts       # 종합소득세 추정 계산
    ├── csvImport.ts         # CSV 파싱
    └── csvExport.ts         # 홈택스 CSV 생성
```

## 로컬 실행

`.env.local` 파일에 Supabase 환경변수 필요:

```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

```bash
npm install
npm run dev
```

빌드:
```bash
npm run build
```
